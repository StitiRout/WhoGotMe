# 🛡️ WhoGotMe

> **A Privacy-Preserving Dark Web Monitoring Simulator for Data Breach Detection**

WhoGotMe is a cybersecurity-focused web application that allows users to check whether their **email address** has been exposed in known data breaches using a **simulated breach database**. The project is inspired by modern breach monitoring platforms while emphasizing **user privacy**, **secure coding practices**, and **cybersecurity awareness**.

---

## 📌 Project Overview

Every year, millions of user records are exposed through data breaches. Many users remain unaware that their personal information has been compromised.

**WhoGotMe** provides a safe simulation of a breach monitoring platform by allowing users to search for their email address against a simulated breach dataset. Instead of using real leaked data, the project uses a controlled dataset for educational and demonstration purposes.

The application processes user input **temporarily in memory** and **does not store** search queries, ensuring user privacy.
